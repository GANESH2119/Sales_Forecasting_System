# Sales Forecasting System

## Project Overview
This project predicts future sales using Machine Learning techniques. The dataset is preprocessed, analyzed, and used to train a prediction model.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Features
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Sales Prediction
- Model Evaluation

## Dataset
Sales Forecasting Dataset
